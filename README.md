# cfe-ha-os

Repositorio de addons para Home Assistant.

## Addons disponibles

### CFE Portal
Extrae automáticamente los datos de tu cuenta CFE — saldo, periodo, fecha límite, estado del recibo y PDF — y los publica como sensores en Home Assistant via MQTT Discovery.

**Agregar repositorio en HA:**

```
https://github.com/cyoulabel/cfe-ha-os
```

Configuración → Complementos → Tienda → ⋮ → Repositorios
